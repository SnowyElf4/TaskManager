# TaskManager v0.2

Простой менеджер задач на Java с поддержкой регистрации пользователей и привязкой задач к конкретному пользователю.

## Особенности

- Регистрация пользователей
- Создание задач
- Выполнение задач
- Просмотр задач конкретного пользователя
- Каждая задача привязана к пользователю, который её создал

## Установка и запуск

1. Клонируйте репозиторий:

```
git clone https://github.com/SnowyElf4/TaskManager.git
```

2. Перейдите в папку проекта:

```
cd TaskManager
```

3. Скомпилируйте проект:

```
javac -d out $(find . -name "*.java")
```

4. Запустите приложение:

```
java -cp out TaskManager.Main
```

## Использование

1. При запуске появится меню для регистрации нового пользователя.
2. После регистрации вы попадёте в главное меню, где можно:

   * Создать новую задачу
   * Отметить задачу как выполненную
   * Просмотреть все свои задачи

Каждый пользователь видит только свои задачи.

## Пример работы

1. Создать пользователя.
Введите ваше имя: SnowyElf
User: SnowyElf (ID: 18881888)

1. Создать задачу.
Введите название задачи: Уборка
Введите описание: Протереть пыль в комнате
Задача создана. ID: 10101010, Название: Уборка

3. Показать все задачи
ID: 10101010, Название: Уборка, Выполнена: false
