# TaskManager v0.3

Консольный менеджер задач на Java с поддержкой нескольких пользователей и постоянным хранением данных в JSON-файлах.

Проект демонстрирует базовую архитектуру приложения с разделением логики, хранения данных и пользовательского интерфейса.

## Возможности

* Регистрация пользователей
* Переключение между пользователями
* Создание задач
* Отметка задач как выполненных
* Просмотр задач конкретного пользователя
* Постоянное хранение пользователей и задач в JSON
* Каждая задача жёстко привязана к пользователю

## Архитектура (кратко)

* `ui` – консольное меню и ввод пользователя
* `service` – бизнес-логика и работа с хранилищем
* `StorageService` – единая точка работы с JSON-файлами
* `user` / `task` – модели данных

Данные не теряются между запусками приложения.

## Установка и запуск

1. Клонировать репозиторий:

```
git clone https://github.com/SnowyElf4/TaskManager.git
```

2. Перейти в директорию проекта:

```
cd TaskManager
```

3. Скомпилировать проект:

```
javac -d out $(find . -name "*.java")
```

4. Запустить приложение:

```
java -cp out TaskManager.Main
```

## Использование

При запуске приложения:

* Если пользователей нет – предлагается создать нового
* Если пользователи существуют – можно выбрать активного пользователя

В главном меню доступны действия:

* создание задачи
* завершение задачи
* просмотр всех своих задач
* смена пользователя

Каждый пользователь видит **только свои задачи**.

## Пример работы

Создание пользователя:

```
Введите имя: SnowyElf
Пользователь создан: SnowyElf (ID: 18881888)
```

Создание задачи:

```
Введите название задачи: Уборка
Введите описание: Протереть пыль в комнате
Задача создана.
ID: 10101010
```

Просмотр задач:

```
ID: 10101010, Название: Уборка, Выполнена: false
```
